# Knowledge Loader Demo

This example demonstrates all built-in knowledge loaders from the
`pkg/agentgo/knowledge` package: text, directory, PDF, CSV, JSON, HTML, URL,
multi-URL, and reader loaders, plus the chunking primitives used to split
documents for vector indexing.

## Prerequisites

- Go 1.24+
- No API key required

## How to Run

```bash
go run ./cmd/examples/knowledge_loader_demo/
```
